Plugins used:
Featherlight: This plugin is used to append lightboxes to various elements
Simpler-Sidebar: This plugin is used to generate sidebars that contain some content
Glowspot: This plugin is used to highlight some content when the cursor hovers on the element
Particleground: This plugin is used to display the animation on the header

Above and beyond:
- All the data from the api was extracted only once
- Various plugins that improve the user's experience of the website were used
- The code written is modular and reusable
